# Strike & Shield

Statisches Dashboard eines vollautomatischen US-Momentum-Signalsystems — zwei
simulierte Depots (Strike/Shield), täglich nach US-Börsenschluss und kurz nach
Eröffnung automatisch aktualisiert.

**Live:** https://rayyanmch.github.io/strike-shield/

Simulation mit echten Marktkursen · keine Anlageberatung · kein Versprechen.
